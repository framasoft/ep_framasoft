Crédits

Ce site utilise des éléments libres communs à l'ensemble du réseau Framasoft :

  • la librairie jQuery
  • le framework libre Bootstrap
  • les polices d'écriture PTSans (titres), Deja Vu Sans et Font-Awesome (icônes)
  • le lecteur vidéo HTML5 VideoJS
  • l'outil de mesure statistique Piwik

Ces éléments sont déployés via la barre de navigation dont le code source peut être consulté sur notre dépôt Gitlab.
